## 📌 About the Project
This project is developed as part of the Operating Systems course at Nile University. It implements a file system utility inspired by *Timeshift*, designed to save the current state of a file system by creating reliable snapshots and providing the ability to restore the system to a previous state later. 

This tool acts as a safeguard, allowing users to revert their environment after unintended changes, system updates, or errors.

## ✨ Core Features (Planned)
* **Snapshot Creation:** Capture the exact state of the file system at a given point in time.
* **System Restoration:** Revert the file system back to a specifically selected snapshot safely.
* **Snapshot Management:** View, list, and delete existing snapshots to manage storage space.
